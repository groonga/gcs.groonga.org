# Install

## Mac OS X

    curl -O http://packages.groonga.org/macosx/gcs/gcs-1.0.0.tar.gz
    tar xvzf gcs-1.0.0.tar.gz
    cd gcs-1.0.0
    ./gcs

## Debian

    sudo apt-get install groonga npm
    sudo npm install -g gcs
    gcs

## CentOS

    sudo yum install groonga
    # How to install npm?
    sudo npm install -g gcs
    gcs
